Authentication

Secure user authentication and session management

POST /api/auth/login

Authenticate user and receive access token

Request Body:

{
  "email": "user@example.com",
  "password": "password123"
}

Response:

{
  "token": "jwt_token_here",
  "user": {
    "id": "user_id",
    "email": "user@example.com"
  }
}
POST /api/auth/register

Create new user account

Request Body:

{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "password123"
}

Response:

{
  "message": "User created",
  "userId": "new_user_id"
}
POST /api/auth/refresh

Refresh expired access token

Whiteboard API

Manage collaborative whiteboard sessions

GET /api/whiteboard/{sessionId}

Retrieve whiteboard session data and drawing elements

POST /api/whiteboard/create

Create new whiteboard session

Request Body:

{
  "name": "Project Meeting",
  "isPrivate": false,
  "maxUsers": 10
}

Response:

{
  "sessionId": "wb_123456",
  "joinUrl": "/whiteboard/wb_123456"
}
PUT /api/whiteboard/{sessionId}/update

Update whiteboard settings or add drawing elements

DELETE /api/whiteboard/{sessionId}

Delete whiteboard session and all associated data

AI Services

Intelligent features powered by AI

POST /api/ai/ocr

Extract text from images using OCR

Request Body:

{

  "language": "en"
}

Response:

{
  "text": "Extracted text",
  "confidence": 0.95
}
POST /api/ai/shape-detection

Detect and convert hand-drawn shapes to perfect geometric shapes

POST /api/ai/summarize

Generate AI-powered summaries of whiteboard content

WebSocket Events

Real-time communication events

drawing_update

Broadcast drawing actions to all connected users

cursor_move

Share cursor position with other participants

user_join

Notify when a user joins the session

user_leave

Notify when a user leaves the session

chat_message

Real-time chat messages between users

Code Examples

Ready-to-use integration examples

JavaScript WebSocket Connection

const ws = new WebSocket('ws://localhost:8080');

ws.onopen = () => {
    console.log('Connected to CollabMeet');
    ws.send(JSON.stringify({
        type: 'join_session',
        sessionId: 'your_session_id'
    }));
};

ws.onmessage = (event) => {
    const data = JSON.parse(event.data);
    if (data.type === 'drawing_update') {
        // Handle drawing updates
        updateCanvas(data.drawingData);
    }
};

API Authentication Example

fetch('/api/auth/login', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        email: 'user@example.com',
        password: 'password123'
    })
})
.then(response => response.json())
.then(data => {
    localStorage.setItem('token', data.token);
    console.log('Logged in successfully');
});
